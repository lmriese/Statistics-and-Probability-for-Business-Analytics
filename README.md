# Statistics and Probability for Business Analytics

A Python-based business analytics project focused on statistical analysis, probability concepts, and retail customer purchasing behavior using Walmart transaction data.

## Project Overview

This project explores customer demographics, purchase behavior, and spending trends using the Walmart retail dataset. The notebook applies exploratory data analysis (EDA), statistical techniques, and data visualization to generate business insights and support data-driven decision-making.

## Features
- Data preprocessing and cleaning
- Exploratory data analysis (EDA)
- Statistical analysis
- Probability-based analysis
- Customer purchasing behavior analysis
- Retail business insights
- Data visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- SciPy
- Statsmodels
- Jupyter Notebook

## Topics Covered
- Descriptive statistics
- Customer demographics analysis
- Purchase distribution analysis
- Retail analytics
- Business intelligence
- Data visualization
- Spending pattern analysis

## Dataset Information
The dataset contains Walmart retail transaction records including:
- Customer demographics
- Product categories
- City categories
- Occupation data
- Purchase amounts
- Customer purchasing behavior
